 Proje Hakkında 

Bu proje, React Native'de ileri seviye stil tekniklerini uygulamayı amaçlamaktadır.Amaç,belirlenen fontlar, tema , duyarlı düzen, çapraz platform gölgeler ve etkileşimli bileşenler kullanılarak profesyonel bir profil kartı uygulaması oluşturmaktır.


Node.js ve npm yüklenmeli
expo go yüklenmeli

Kurulum
Bağımlılıklar

    npm install

Başlatma

    npx expo start
